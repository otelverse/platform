# TurtleBot3 Observability Example

This example demonstrates how to integrate the OTelVerse Robotics SDK into a TurtleBot3 Gazebo simulation.
To run:
1. `docker-compose up -d` in `packages/robotics-sdk/docker/simulation-env`
2. Run `ros2 run teleop_twist_keyboard teleop_twist_keyboard`
3. View traces in the Platform.
